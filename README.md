# Orman Yangını Erken Uyarı Sistemi

Bu proje, orman yangınlarını başlamadan önce tahmin etmeyi, risk bölgelerini belirlemeyi ve yetkililere otomatik uyarı göndermeyi hedefleyen bir erken uyarı platformudur.

## Başlangıç

Sistemi ayağa kaldırmak için `docker-compose` kullanabilirsiniz:

```bash
docker-compose up -d --build
```

Daha fazla detay için projenin ana dökümanını (Prompt) inceleyin.
